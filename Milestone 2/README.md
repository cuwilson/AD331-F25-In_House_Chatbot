## Deliverables & Submission Requirements

### Submission Format
A **4–6 page document** submitted as a **PDF** or **DOCX** file.

### Required Sections
- **Detailed System Architecture Diagram**  
  Show the data flow from input → model → response → storage.
  
- **Component Descriptions**  
  Explain how each part functions (frontend, backend, model, storage, authentication).

- **Security and Data Protection Plan**  
  Outline authentication, access control, encryption, and restricted data handling.

- **Technology and Tool Justification**  
  Explain the rationale behind each chosen technology.

- **Refined Requirements List**  
  Incorporate and adjust based on feedback from **Milestone 1**.
